## Scope

This doc tree covers:

- order runtime and distribution architecture
- order-specific roadmap and validation criteria
- order plugin install and runtime usage guidance
- order-only design reference and development logs

It does not replace:

- the runtime README under `order/`
- the plugin README under `plugins/openclaw-order/`
- test and migration scripts under `scripts/`
